# File Format Specifications
Specifications for properietary file formats, binary structures and other systens created and used by Level5 in the Yo-kai Watch Series. This is intended for a technical audience and *not* the majority of modders. 
It is recommended to start on the [Signature](./file-formats/level5-sigs.html) and [Level-5 Compression Container](./file-formats/level5-compressed-container.html) pages.
