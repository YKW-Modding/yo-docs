# Downloading CfgBinEditor Tags
Step 1. [Click this link](https://raw.githubusercontent.com/light8227/ykw-stuff/refs/heads/master/MyTags.txt). Then click `Ctrl + S`. This will save the `MyTags.txt` file.
Step 2. Place the file next to your CfgBinEditor's `.exe`.
Step 3. Close and Reopen CfgBinEditor if it's already open
